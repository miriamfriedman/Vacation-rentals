# Vacation Rentals

A final project system for publishing and managing vacation rental apartments.

## Project Description

This system allows advertisers to publish vacation apartments while users can browse, search, and filter listings by categories and cities.

The project was built as a final school project.

## Technologies Used

### Frontend

* React
* JavaScript (ES6+)
* CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## Project Structure

### finallnode-react

The frontend application (client side).

### finallnode-node

The backend API server.

## Installation & Running

### Backend

```bash
cd finallnode-node
npm install
npm start
```

### Frontend

```bash
cd finallnode-react
npm install
npm start
```

## Features

* Manage vacation apartments
* Manage advertisers
* Manage categories
* Manage cities
* Full REST API communication between client and server
* Dynamic filtering and search functionality
